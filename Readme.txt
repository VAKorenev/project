Задача:
Поступает задача, которую в работу берет руководитель проекта (РП). Он заносит все данные в разрабатываемую систему управления проектами (СУП == БОРЩ).
Основаная задача СУПа на первый этап - посчитать:  
* затраты по проекту;
* доходы по проекту;
* денежное вознаграждение участникам (они на сделке);
* итоговый подробный отчет по проекту(ам);
